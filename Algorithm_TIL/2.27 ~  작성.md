#### [2/27] 이코테 page114
- 문자열을 완전탐색하는 문제인데 중요한 것은 str() + str() 의 형식으로 문자열을 이어 붙일 수 있다는 것과 문자열을 'in' 키워드를 이용해서 특정
문자열 내에 내가 찾고자 하는 문자가 있는지 확인이 가능하다는 것을 기억하자!
```
n = int(input())
count = 0
for i in range(0, n+1):
  for j in range(0, 60):
    for k in range(0, 60):
      full = str(i) + str(j) + str(k) 
      if '3' in full:
        count += 1

print(count)
```
